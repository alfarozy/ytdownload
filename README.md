# ytdownload

Download video & playlist youtube menggunakan terminal di linux.

## requirements

1. python3
2. pytube3 - pip3 install pytub


## install

1. clone this repo
2. cd ytdownload
3. chmod +x ytdownload
4. python3 ytdownload
5. klik h for help
